# SOHMA AI

> **Emotional Intelligence Infrastructure for Adaptive AI Systems**

SOHMA builds emotional intelligence infrastructure for AI systems, enabling adaptive AI through behavioural signals, human-AI interaction, and real-world environments. We are focused on AI safety, responsible AI, and interaction design across gaming, education, and wellbeing.

## Overview

This repository contains the static landing page and website for **SOHMA AI**. It serves as the primary informational hub for our mission, technology, and collaboration opportunities. 

Artificial intelligence has made enormous progress in reasoning and language, but most systems still struggle to interpret the behavioural and emotional signals that shape human decision-making. SOHMA acts as an infrastructure layer that sits between user interaction and AI decision-making, enabling systems to understand and respond safely to human emotional regulation.

## How it Works

SOHMA transforms human interaction into structured behavioural signals, connecting interaction data, behavioural interpretation, and adaptive response orchestration into a unified system layer:

1. **Human Interaction:** Users engage with AI systems across games, learning platforms, wellbeing tools, and conversational environments.
2. **Behavioural Signals:** Interaction produces continuous signals such as hesitation, pacing, engagement, disengagement, escalation, and recovery.
3. **SOHMA Layer:** These signals are structured into interpretable indicators that represent the user's behavioural and emotional context.
4. **Adaptive Behaviour:** AI systems can then adjust their pacing, support, challenge level, and interaction style in real-time response to that context.

## Core Capabilities

- **Behavioural Intelligence Layer:** SOHMA processes interaction dynamics to build a real-time understanding of user state. The structured, interpretable outputs can be used by AI systems to adapt responses, adjust difficulty, or trigger interventions.
- **Infrastructure for AI Systems:** By acting as a middle layer, developers can integrate behavioural intelligence into their existing systems *without* modifying their core models. SOHMA provides the structured signals, control layers, and transparent outputs necessary for human-aware system design.

## Development & Deployment

The website is entirely static and designed to be hosted directly via GitHub Pages (or any static hosting provider). There is no build step or server-side configuration required. 

To run the site locally for development or preview:
1. Clone this repository.
2. Open `index.html` in your browser, or start a local HTTP server:
   ```bash
   python3 -m http.server 8080
   ```
3. Navigate to `http://localhost:8080`.

### Hosting on GitHub Pages
This repository is configured to work out-of-the-box with GitHub Pages. Simply enable GitHub Pages in your repository settings and point it to the `main` branch (or the root directory). The `404.html` file will automatically intercept and handle any missing routes.
